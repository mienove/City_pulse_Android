# Mini-Projet Android
## CityPulse - Explorez votre ville en temps réel

---

## Présentation du projet

CityPulse est une application Android développée dans le cadre d’un mini-projet académique.  
Elle permet aux utilisateurs d’explorer leur environnement en temps réel grâce à une carte interactive, une liste dynamique de lieux et un système de favoris.

L’application combine des technologies modernes du développement Android telles que :

- Google Maps SDK
- API REST (Retrofit)
- Base de données locale (Room)
- Architecture MVVM
- Mode hors-ligne

---

## Objectifs du projet

- Permettre la découverte de lieux proches de l’utilisateur
- Fournir une interface intuitive et moderne
- Offrir un mode hors connexion via une base locale
- Implémenter une gestion des favoris
- Appliquer les bonnes pratiques du développement Android

## Fonctionnalités de l’application

### Carte interactive
- Affichage de la localisation de l’utilisateur
- Marqueurs des lieux proches en temps réel
- Navigation fluide avec Google Maps

### Liste des lieux
- Affichage des lieux disponibles
- Filtrage par catégorie (restaurant, parc, café, etc.)
- Recherche dynamique

### Système de favoris
- Ajout de lieux favoris
- Stockage local avec Room
- Accès hors connexion
- Suppression par swipe

### Mode sombre / clair
- Support automatique du thème système
- Interface adaptée aux préférences utilisateur
